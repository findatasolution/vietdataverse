Lấy dữ liệu vào Excel (Power Query)

Kéo dữ liệu tài chính Việt Nam — vàng, bạc, tỷ giá, lãi suất, CPI, dữ liệu toàn cầu — thẳng vào Microsoft Excel bằng Power Query, không cần code. Dữ liệu nạp thành Table và làm mới được bằng một nút bấm.

Bạn sẽ làm được gì

Excel có sẵn Power Query (Get & Transform Data) — tương đương IMPORTDATA của Google Sheets nhưng mạnh hơn: đọc được cả CSV lẫn JSON, nạp vào Table, và làm mới tự động.

Không cần cài thêm gì. Power Query có sẵn trong Excel 2016 trở lên (Windows & Mac) và Microsoft 365. Khác với Google Sheets, Excel không hiện cảnh báo "ứng dụng chưa xác minh".

Bước 1 — Lấy API token

1
Mở trang Developer rồi đăng nhập (đăng ký miễn phí nếu chưa có).
2
Tài khoản miễn phí được cấp ngay API key với hạn mức 1.000 request/tháng.
3
Nhấn "Sao chép" để lấy key đầy đủ (~43 ký tự). Giữ key bí mật.

Bước 2 — Kết nối bằng Power Query

1
Trong Excel: thẻ Data → Get Data → From Other Sources → From Web (hoặc Data → From Web).
2
Dán URL của bộ dữ liệu (thay YOUR_API_KEY bằng token). Ví dụ giá vàng SJC 1 năm:
https://api.vietdataverse.online/api/v1/gold?type=SJC&period=1y&page=1&limit=500&api_key=YOUR_API_KEY
3
Nếu Excel hỏi cách xác thực → chọn Anonymous → Connect (token đã nằm trên URL).
4
Cửa sổ Power Query mở ra:
  • Dữ liệu JSON (bạc, lãi suất, CPI…): click vào trường data → To Table → nhấn nút mở rộng (↔) ở header cột để tách thành các cột (date, giá trị…).
  • Giá vàng dạng CSV: nếu bạn bỏ page và thêm &format=csv, Excel nhận luôn dạng bảng, không cần expand.
5
Nhấn Close & Load → dữ liệu đổ vào sheet thành một Table.
Mẹo: Tham số &page=1&limit=500 giúp API trả về dạng danh sách dòng — Power Query expand thẳng thành bảng sạch.

Bước 3 — Làm mới dữ liệu

Cập nhật bất cứ lúc nào: Data → Refresh All. Tự động:

  1. Chuột phải vào Table → Table → External Data Properties (hoặc Data → Queries & Connections → Properties).
  2. Tick "Refresh data when opening the file" và/hoặc "Refresh every N minutes".
Mỗi lần refresh = 1 request tính vào quota tháng. Đừng đặt refresh quá dày trên nhiều query cùng lúc.

Danh sách bộ dữ liệu

Tất cả các bộ dữ liệu có trên trang chủ Viet Dataverse đều dùng được trong Excel. Ghép tham số vào URL: ?period=...&page=1&limit=500&api_key=YOUR_API_KEY

Bộ dữ liệuEndpointTham số chínhCột trả về
Giá vàng /api/v1/gold type = SJC · DOJI HN · PNJ · BTMC; period date · buy_price · sell_price
Giá bạc /api/v1/silver period date · buy_price · sell_price
Tỷ giá ngoại tệ (VCB) /api/v1/sbv-rate bank = VCB; currency = USD · EUR · JPY…; period date · buy · buy_cash · sell
Lãi suất liên ngân hàng (SBV) /api/v1/sbv-interbank period dates · overnight · month_1 · month_3 · month_6…
Lãi suất tiết kiệm /api/v1/termdepo period date · term_1m · term_3m · term_6m · term_12m · term_24m
CPI / Lạm phát /api/v1/macro/cpi period period · yoy_pct · months
Dữ liệu toàn cầu /api/v1/global-macro period date · gold_price · silver_price · nasdaq_price

period nhận: 7d · 1m · 1y · all. Ví dụ lấy lãi suất tiết kiệm 1 năm:

https://api.vietdataverse.online/api/v1/termdepo?period=1y&page=1&limit=500&api_key=YOUR_API_KEY
Danh sách endpoint đầy đủ (kèm VN30, news…) xem ở API Reference.

Lỗi thường gặp

Báo lỗi 401 / "Access to the resource is forbidden"

Token sai hoặc đã bị thu hồi. Lấy lại key mới nhất ở trang Developer (mỗi lần "Generate New Key" sẽ thu hồi key cũ).

Power Query hỏi đăng nhập (Organizational / Windows…)

Chọn Anonymous. Token đã đi trên URL nên không cần xác thực kiểu khác.

Dữ liệu JSON nằm trong 1 cột data

Đúng quy trình — click To Table rồi nhấn nút mở rộng (↔) ở header để tách cột. Hoặc thêm &page=1&limit=500 để API trả dạng danh sách dòng cho dễ expand.
