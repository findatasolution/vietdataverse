Quick Start

Gọi API trong 60 giây với X-API-Key header.

curl https://api.vietdataverse.online/api/v1/gold?period=1m \ -H "X-API-Key: YOUR_API_KEY"
import requests headers = {"X-API-Key": "YOUR_API_KEY"} resp = requests.get( "https://api.vietdataverse.online/api/v1/gold", params={"period": "1m"}, headers=headers, ) data = resp.json() # {"success": true, "count": 30, "data": [...]}
const res = await fetch( "https://api.vietdataverse.online/api/v1/gold?period=1m", { headers: { "X-API-Key": "YOUR_API_KEY" } } ); const { data } = await res.json();

Authentication

Mọi request cần header X-API-Key với key nhận được sau khi mua gói Developer.

X-API-Key: vdv_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

Lấy API key tại trang Developer sau khi đăng ký gói. Key được liên kết với tài khoản — không chia sẻ key để tránh bị tính quota.

Quota & Giới hạn

Quota reset vào ngày 1 hàng tháng (múi giờ Asia/Ho_Chi_Minh). Burst limit bảo vệ hệ thống.

Header phản hồi: X-RateLimit-Limit, X-RateLimit-Remaining, X-RateLimit-Reset (ISO 8601).
