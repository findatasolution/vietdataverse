üí° Quick Summary

XGBoost (Extreme Gradient Boosting) is an optimized distributed gradient boosting library designed to be highly efficient, flexible, and portable. It implements machine learning algorithms under the Gradient Boosting framework and has become one of the most popular ML algorithms due to its performance and speed.

Why XGBoost is Popular

  • Speed & Performance - Optimized for computational efficiency
  • Regularization - Built-in L1 and L2 regularization
  • Handling Missing Values - Automatic handling of sparse data
  • Parallel Processing - Utilizes multiple CPU cores
  • Cross-Validation - Built-in CV functionality

Key Components

  • Objective Function = Loss Function + Regularization Term
  • Gradient & Hessian - Second-order approximation for optimization
  • Tree Pruning - Max depth constraint and pruning based on split gain
  • Learning Rate (Œ∑) - Shrinkage to prevent overfitting
