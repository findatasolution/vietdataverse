üí° Quick Summary

Decision Trees are supervised learning algorithms used for both classification and regression tasks. They work by recursively splitting the data based on feature values to create a tree-like structure of decisions that lead to predictions.

Key Advantages

  • Easy to understand and interpret (white-box model)
  • Requires little data preparation
  • Can handle both numerical and categorical data
  • Non-parametric (no assumptions about data distribution)

Key Challenges

  • Prone to overfitting (especially deep trees)
  • Can be unstable (small data changes ‚Üí different tree)
  • Biased toward features with more levels
