1. Introduction to Functions

1.1 Definition

A function $f$ from set $A$ to set $B$ (written $f: A \to B$) is a rule that assigns to each element $a \in A$ exactly one element $b \in B$.

Notation:

  • $f: A \to B$ means "$f$ is a function from $A$ to $B$"
  • $f(a) = b$ means "$f$ maps element $a$ to element $b$"
  • $A$ is called the domain
  • $B$ is called the codomain
  • The range is the set $\{f(a) : a \in A\}$

1.2 Function Properties

Functions can be classified based on how they map elements:

One-to-One (Injective)

Definition: Different inputs map to different outputs

$$f(a_1) = f(a_2) \implies a_1 = a_2$$

Example: $f(x) = 2x$ is one-to-one

Onto (Surjective)

Definition: Every element in codomain is mapped to

$$\forall b \in B, \exists a \in A: f(a) = b$$

Example: $f: \mathbb{R} \to \mathbb{R}, f(x) = x^3$ is onto

Bijection (One-to-One Correspondence)

A function that is both one-to-one and onto

  • Every element in domain maps to unique element in codomain
  • Every element in codomain is mapped to
  • Bijections have inverse functions

2. Function Examples & Diagrams

2.1 Arrow Diagrams

Arrow diagrams provide a visual way to represent functions, especially useful for finite sets.

Example 2.1: Function from $A = \{1, 2, 3\}$ to $B = \{a, b, c\}$

Let $f: A \to B$ where:

  • $f(1) = a$
  • $f(2) = b$
  • $f(3) = c$

Analysis:

  • One-to-one: Each element maps to different output
  • Onto: All elements of $B$ are covered
  • Bijection: Both properties hold
Example 2.2: Non-injective Function

Let $g: \{1, 2, 3\} \to \{a, b\}$ where:

  • $g(1) = a$
  • $g(2) = a$
  • $g(3) = b$

Analysis:

  • Not one-to-one: $g(1) = g(2) = a$ but $1 \neq 2$
  • Onto: Both $a$ and $b$ are mapped to

2.2 Testing Function Properties

How to check if $f: A \to B$ is one-to-one:

  1. Assume $f(a_1) = f(a_2)$ for some $a_1, a_2 \in A$
  2. Show that this implies $a_1 = a_2$
  3. If this holds for all elements, $f$ is one-to-one

How to check if $f: A \to B$ is onto:

  1. Take an arbitrary element $b \in B$
  2. Find an element $a \in A$ such that $f(a) = b$
  3. If this can be done for every $b \in B$, $f$ is onto

3. Function Composition

3.1 Definition

If $f: A \to B$ and $g: B \to C$, the composition $g \circ f: A \to C$ is defined by:

$$(g \circ f)(a) = g(f(a))$$
Important: The order matters! $g \circ f$ means "apply $f$ first, then $g$"
Example 3.1: Composition of Functions

Let $f: \mathbb{R} \to \mathbb{R}$ with $f(x) = 2x + 1$

Let $g: \mathbb{R} \to \mathbb{R}$ with $g(x) = x^2$

Find $(g \circ f)(x)$:

$$ (g \circ f)(x) = g(f(x)) = g(2x + 1) = (2x + 1)^2 = 4x^2 + 4x + 1 $$

Find $(f \circ g)(x)$:

$$ (f \circ g)(x) = f(g(x)) = f(x^2) = 2x^2 + 1 $$

Notice: $(g \circ f)(x) \neq (f \circ g)(x)$ in general!

3.2 Properties of Composition

  • If $f$ and $g$ are both one-to-one, then $g \circ f$ is one-to-one
  • If $f$ and $g$ are both onto, then $g \circ f$ is onto
  • If $f$ and $g$ are both bijections, then $g \circ f$ is a bijection
  • Composition is associative: $(h \circ g) \circ f = h \circ (g \circ f)$

4. Inverse Functions

4.1 Definition

If $f: A \to B$ is a bijection, the inverse function $f^{-1}: B \to A$ satisfies:

$$f^{-1}(b) = a \iff f(a) = b$$ $$f^{-1}(f(a)) = a \text{ for all } a \in A$$ $$f(f^{-1}(b)) = b \text{ for all } b \in B$$
Key Fact: A function has an inverse if and only if it is a bijection.
Example 4.1: Finding Inverse Function

Let $f: \mathbb{R} \to \mathbb{R}$ with $f(x) = 3x - 5$

Step 1: Verify $f$ is a bijection

  • One-to-one: If $3x_1 - 5 = 3x_2 - 5$, then $x_1 = x_2$
  • Onto: For any $y \in \mathbb{R}$, we can find $x = \frac{y + 5}{3}$ such that $f(x) = y$

Step 2: Find $f^{-1}$

Let $y = 3x - 5$, solve for $x$:

$$ y = 3x - 5 \\ y + 5 = 3x \\ x = \frac{y + 5}{3} $$

Therefore: $f^{-1}(y) = \frac{y + 5}{3}$

Verification:

$$ f(f^{-1}(y)) = f\left(\frac{y+5}{3}\right) = 3\cdot\frac{y+5}{3} - 5 = y $$

5. Special Types of Functions

5.1 Hash Functions

A hash function maps data of arbitrary size to fixed-size values. In discrete mathematics, we often use modular arithmetic:

$$h(x) = x \bmod m$$
Example 5.1: Simple Hash Function

Let $h: \mathbb{Z} \to \{0, 1, 2, 3, 4\}$ where $h(x) = x \bmod 5$

  • $h(7) = 7 \bmod 5 = 2$
  • $h(12) = 12 \bmod 5 = 2$
  • $h(23) = 23 \bmod 5 = 3$

Note: This is not one-to-one (collision: $h(7) = h(12)$)

ML Application: Hash functions are used in:
  • Feature hashing: Map high-dimensional features to fixed size
  • Data structures: Hash tables for fast lookup
  • Cryptography: Secure hash functions in blockchain

5.2 Floor and Ceiling Functions

Floor function: $\lfloor x \rfloor$ = largest integer $\leq x$

Ceiling function: $\lceil x \rceil$ = smallest integer $\geq x$

Examples:

  • $\lfloor 3.7 \rfloor = 3$, $\lceil 3.7 \rceil = 4$
  • $\lfloor -2.3 \rfloor = -3$, $\lceil -2.3 \rceil = -2$
  • $\lfloor 5 \rfloor = 5$, $\lceil 5 \rceil = 5$

6. Practice Problems

Practice 1: Determine Function Properties

For each function, determine if it is one-to-one, onto, or bijection:

a) $f: \mathbb{Z} \to \mathbb{Z}$, $f(x) = x + 3$

Solution:

  • One-to-one: If $x_1 + 3 = x_2 + 3$, then $x_1 = x_2$
  • Onto: For any $y \in \mathbb{Z}$, let $x = y - 3$, then $f(x) = y$
  • Bijection: Yes

b) $g: \mathbb{Z} \to \mathbb{Z}$, $g(x) = x^2$

Solution:

  • One-to-one: $g(-2) = g(2) = 4$ but $-2 \neq 2$, so NO
  • Onto: No integer $x$ satisfies $x^2 = -1$, so NO
  • Bijection: NO

c) $h: \mathbb{N} \to \mathbb{N}$, $h(x) = 2x$

Solution:

  • One-to-one: If $2x_1 = 2x_2$, then $x_1 = x_2$, so YES
  • Onto: No natural number maps to $3$ (odd numbers not in range), so NO
  • Bijection: NO

Practice 2: Function Composition

Let $f(x) = x^2 + 1$ and $g(x) = 3x - 2$

a) Find $(f \circ g)(x)$

Solution:

$$ (f \circ g)(x) = f(g(x)) = f(3x - 2) = (3x - 2)^2 + 1 = 9x^2 - 12x + 5 $$

b) Find $(g \circ f)(x)$

Solution:

$$ (g \circ f)(x) = g(f(x)) = g(x^2 + 1) = 3(x^2 + 1) - 2 = 3x^2 + 1 $$

Practice 3: Inverse Functions

Find the inverse of $f(x) = \frac{2x - 3}{5}$ (if it exists)

Solution:

Step 1: Set $y = \frac{2x - 3}{5}$ and solve for $x$:

$$ y = \frac{2x - 3}{5} \\ 5y = 2x - 3 \\ 5y + 3 = 2x \\ x = \frac{5y + 3}{2} $$

Therefore: $f^{-1}(y) = \frac{5y + 3}{2}$

Verification:

$$ f(f^{-1}(y)) = f\left(\frac{5y+3}{2}\right) = \frac{2\cdot\frac{5y+3}{2} - 3}{5} = \frac{5y + 3 - 3}{5} = y $$
